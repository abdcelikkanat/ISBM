# Structure

This repository is organized into three folders:

1. **BuildingBlocks**: Contains the core “building blocks” and underlying concepts used in the larger algorithms.
2. **Implementations**: Contains `.ipynb` notebooks where I describe and implement the methods from the paper.
3. **src**: Contains the combined implementations from the `Implementations` folder as plain `.py` files without the markdown explanations.

My first implementation of the CGS called `CGS_1` is not the optimal implementation, and it has a couple of errors and suboptimal solutions, but I chose to keep it for reference.
